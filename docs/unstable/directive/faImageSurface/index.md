<h1 class="api-title">

  fa-image-surface



</h1>





This directive creates a Famo.us ImageSurface and loads
the specified ImageUrl.








  
<h2 id="usage">Usage</h2>
  
```html
<fa-image-surface fa-image-url="img/my-image.png">
</fa-image-surface>
```
